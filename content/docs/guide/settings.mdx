# 设置说明

OuraPix 提供丰富的设置选项，让您可以根据需求定制详情页生成体验。

## 账户设置

### 个人资料

在「账户设置」中可以修改：
- 用户名
- 邮箱地址
- 密码
- 头像

### 订阅管理

查看当前订阅计划：
- 免费版：每月 10 次生成
- 专业版：每月 100 次生成
- 企业版：无限次生成 + API 访问

## 生成设置

### 默认平台

设置您常用的电商平台：
- **Amazon** - 针对 A+ 内容优化
- **Shopify** - 适合独立站风格
- **Temu** - 符合平台规范
- **通用** - 适配多平台

### 默认语言

选择常用输出语言：
- 英语（美国/英国）
- 日语
- 德语
- 法语
- 西班牙语
- 意大利语

### 文案风格

自定义 AI 生成文案的风格：

| 风格 | 特点 | 适用场景 |
|------|------|----------|
| 专业 | 正式、详细 | B2B 产品 |
| 活泼 | 轻松、有趣 | 年轻消费群体 |
| 简洁 | 简短、直接 | 快消品 |
| 奢华 | 高端、优雅 | 奢侈品 |

## 高级设置

### API 密钥

企业版用户可以生成 API 密钥，集成 OuraPix 到自己的系统中。

```bash
# API 调用示例
curl -X POST https://api.ourapix.com/v1/generate \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "product_name": "Wireless Headphones",
    "platform": "amazon",
    "language": "en"
  }'
```

### 品牌设置

上传您的品牌资料：
- Logo
- 品牌色彩
- 品牌语调
- 常用短语

启用品牌设置后，AI 会在生成内容时保持品牌一致性。

### 模板管理

保存常用的生成配置为模板：
1. 配置好生成参数
2. 点击「保存为模板」
3. 命名模板
4. 下次直接使用

## 通知设置

### 邮件通知

选择接收的通知类型：
- 生成完成通知
- 账户余额提醒
- 产品更新动态
- 优惠活动信息

### Webhook（企业版）

配置 Webhook URL 接收实时事件通知：
- `generation.completed` - 生成完成
- `generation.failed` - 生成失败
- `credits.low` - 余额不足

## 隐私设置

### 数据保留

控制生成数据的保留期限：
- 30 天
- 90 天
- 1 年
- 永久

### 数据导出

导出您的所有数据：
- 产品信息
- 生成的详情页
- 使用记录

## 快捷键

提升效率的键盘快捷键：

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + U` | 打开上传 |
| `Ctrl/Cmd + Enter` | 开始生成 |
| `Ctrl/Cmd + D` | 下载当前详情页 |
| `Esc` | 关闭弹窗 |
