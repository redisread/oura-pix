# 下载指南

OuraPix 支持多种格式导出，满足不同平台的使用需求。

## 支持的格式

### 图片格式

| 格式 | 说明 | 适用场景 |
|------|------|----------|
| JPG | 压缩率高，文件小 | 网页展示 |
| PNG | 无损压缩，支持透明 | 需要透明背景 |
| WebP | 现代格式，质量高 | 网页优化 |

### 文档格式

| 格式 | 说明 | 适用场景 |
|------|------|----------|
| HTML | 完整网页代码 | 直接上传到电商平台 |
| Markdown | 纯文本格式 | 二次编辑 |
| PDF | 固定排版文档 | 存档、打印 |
| DOCX | Word 文档 | 团队协作编辑 |

## 下载步骤

### 单条下载

1. 在生成结果页面，点击「下载」按钮
2. 选择需要的格式
3. 选择包含的内容（图片、文案、完整页面）
4. 点击「确认下载」

### 批量下载

1. 在「我的生成」页面，勾选需要下载的项目
2. 点击「批量下载」
3. 选择格式和打包方式
4. 等待打包完成后下载

## 平台特定导出

### Amazon A+ 内容

导出为 Amazon A+ 格式：
- 标准模块布局
- 符合 Amazon 图片规范
- 自动优化关键词

导出文件包含：
```
amazon_export/
├── images/
│   ├── main.jpg
│   ├── feature_1.jpg
│   └── feature_2.jpg
├── a_plus_content.html
└── keywords.txt
```

### Shopify 产品页

导出为 Shopify 兼容格式：
- Liquid 模板代码
- 优化的产品描述
- SEO 元数据

### Temu 详情页

针对 Temu 平台优化：
- 符合平台尺寸要求
- 简化的文案格式
- 规范的文件命名

## 自定义导出

### 自定义尺寸

设置导出图片尺寸：
- Amazon 主图：2000x2000px
- 详情页长图：800x2000px
- 自定义尺寸

### 水印设置

添加品牌水印：
1. 上传水印图片
2. 设置位置和透明度
3. 应用到导出图片

### 压缩选项

调整图片压缩质量：
- 高质量（90%）- 文件较大，画质最佳
- 标准质量（75%）- 平衡选择
- 网页优化（60%）- 文件最小

## 下载管理

### 下载历史

在「下载记录」中查看：
- 下载时间
- 文件格式
- 文件大小
- 下载次数

### 重新下载

生成的内容保存 30 天，期间可以：
- 免费重新下载
- 更换格式下载
- 批量重新下载

### 分享链接

生成分享链接发送给团队成员：
- 设置链接有效期
- 设置访问密码
- 查看下载次数

## API 下载（企业版）

通过 API 获取生成的内容：

```javascript
// 获取生成结果
const response = await fetch(
  'https://api.ourapix.com/v1/generations/GEN_ID/download',
  {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  }
);

const blob = await response.blob();
// 处理下载文件
```

## 常见问题

**Q: 下载的文件打不开？**

确保使用支持该格式的软件打开。HTML 文件建议使用浏览器，DOCX 使用 Microsoft Word 或 WPS。

**Q: 可以修改已下载的文件吗？**

可以。建议下载 Markdown 或 DOCX 格式进行二次编辑。

**Q: 下载次数有限制吗？**

没有限制，您可以在保存期内无限次下载已生成的内容。
