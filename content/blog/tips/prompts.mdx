# AI 提示词技巧：让详情页更出色

提示词（Prompt）的质量直接影响 AI 生成内容的效果。本文分享 OuraPix 团队总结的提示词技巧。

## 提示词基础结构

### 核心公式

```
[角色] + [任务] + [背景] + [要求] + [格式]
```

**示例：**

```
作为跨境电商文案专家（角色），
为一款无线蓝牙耳机撰写产品描述（任务）。
产品面向 25-35 岁都市白领，主打降噪功能和长续航（背景）。
要求突出产品优势，语言简洁有力，适合 Amazon 平台（要求）。
输出格式：5 个 bullet points（格式）。
```

## 产品描述提示词模板

### 模板 1：功能导向

```
产品：[产品名称]
核心功能：
1. [功能 1]
2. [功能 2]
3. [功能 3]

目标用户：[用户画像]
使用场景：[场景描述]

请撰写产品描述，要求：
- 每个功能配一个使用场景
- 使用 FAB 法则（特性-优势-好处）
- 语言风格：[专业/活泼/简洁/奢华]
```

### 模板 2：情感导向

```
产品：[产品名称]
品牌理念：[品牌故事/价值观]
目标情感：[舒适/自信/愉悦/安心]

请撰写产品描述，要求：
- 以故事开场
- 融入情感元素
- 引发用户共鸣
- 结尾有行动号召
```

### 模板 3：对比导向

```
产品：[产品名称]
解决痛点：[现有产品的不足]
独特优势：[与竞品相比的优势]

请撰写产品描述，要求：
- 先描述痛点引发共鸣
- 介绍产品如何解决
- 用数据支撑优势
- 添加社会证明
```

## 高级技巧

### 1. 少样本提示（Few-Shot）

提供示例帮助 AI 理解风格：

```
请参考以下示例风格，为我的产品撰写描述：

示例 1：
"不是普通的保温杯。12 小时保温，让你的咖啡始终保持刚冲泡的温度。
316 不锈钢内胆，无异味，更健康。"

示例 2：
"告别线缆缠绕。真无线设计，取出即连，戴上即听。
单次 8 小时续航，配合充电盒整整 40 小时。"

现在为以下产品撰写：
[产品信息]
```

### 2. 链式思考（Chain of Thought）

引导 AI 分步思考：

```
请按以下步骤撰写产品描述：

步骤 1：分析目标用户的核心需求
步骤 2：列出产品能满足需求的 3 个关键特性
步骤 3：为每个特性撰写一个具体的使用场景
步骤 4：将所有内容整合成流畅的描述
步骤 5：优化标题，使其更吸引人

产品：[产品信息]
```

### 3. 角色扮演

让 AI 扮演特定角色：

| 角色 | 适用场景 | 效果 |
|------|----------|------|
| 资深产品经理 | 技术产品 | 专业、详细 |
| 生活方式博主 | 时尚产品 | 亲切、有感染力 |
| 行业专家 | B2B 产品 | 权威、可信 |
| 用户代表 | 大众产品 | 真实、易懂 |

## 平台特定提示词

### Amazon 优化

```
为 Amazon 平台撰写产品描述，要求：
- 标题不超过 200 字符，包含核心关键词
- 5 个 bullet points，每个以核心特性开头
- 后端关键词包含同义词和长尾词
- 符合 Amazon 风格指南
```

### Shopify 优化

```
为 Shopify 独立站撰写产品描述，要求：
- 有吸引力的 H1 标题
- 分段清晰，使用小标题
- 融入品牌故事
- 添加情感化描述
- 包含技术规格表格
```

### Temu 优化

```
为 Temu 平台撰写产品描述，要求：
- 简洁直接，突出性价比
- 强调产品规格和尺寸
- 使用表情符号增加可读性
- 包含使用提示
```

## 常见错误

### 1. 提示词过于模糊

❌ **错误：**
```
写一个好用的产品描述。
```

✅ **正确：**
```
为一款便携式蓝牙音箱撰写产品描述，目标用户是 20-30 岁的户外爱好者，突出防水和续航特性，语言风格活泼年轻，适合 Instagram 推广。
```

### 2. 要求相互矛盾

❌ **错误：**
```
用 50 个字详细描述产品的所有功能。
```

✅ **正确：**
```
用 50 个字简洁描述产品最核心的 3 个功能。
```

### 3. 缺乏上下文

❌ **错误：**
```
这个产品怎么样？
```

✅ **正确：**
```
分析这款智能手表在健身追踪方面的表现，对比 Apple Watch 和 Garmin 的优势和不足。
```

## 实战案例

### 案例：瑜伽垫

**基础提示词：**
```
为瑜伽垫写产品描述。
```

**优化后提示词：**
```
作为瑜伽用品专家，为一款 TPE 环保瑜伽垫撰写 Amazon 产品描述。

产品特性：
- 材质：TPE，环保无味
- 厚度：6mm，双面防滑纹理
- 尺寸：183cm x 61cm
- 包含：瑜伽垫 + 背带

目标用户：瑜伽初学者和中级练习者

要求：
1. 标题包含关键词 "TPE Yoga Mat Non Slip"
2. 5 个 bullet points，突出防滑、环保、便携
3. 产品描述分段清晰，包含使用场景
4. 语言专业但易懂
```

**效果对比：**
- 基础提示词：生成通用描述
- 优化提示词：生成专业、针对性的描述，转化率提升 30%
