# Amazon 详情页模板指南

Amazon 详情页是影响转化率的关键因素。本文将介绍如何使用 OuraPix 生成符合 Amazon 规范的 A+ 内容。

## Amazon A+ 内容概述

A+ Content（原 EBC，Enhanced Brand Content）允许品牌卖家使用丰富的图文模块展示产品，提升：
- 转化率 3-10%
- 品牌认知度
- 客户信任度

### A+ 内容模块类型

| 模块 | 用途 | 图片尺寸 |
|------|------|----------|
| Standard Company Logo | 品牌 Logo 展示 | 600x180px |
| Standard Single Left Image | 左图右文 | 300x300px |
| Standard Single Right Image | 右图左文 | 300x300px |
| Standard Three Images & Text | 三图并列 | 300x300px x3 |
| Standard Comparison Chart | 产品对比 | 150x150px |

## OuraPix Amazon 模板

### 模板 1：标准产品展示

适合大多数产品的基础模板：

```
[品牌横幅]
    ↓
[主图 + 核心卖点]
    ↓
[功能特性 x3]
    ↓
[使用场景]
    ↓
[产品规格]
    ↓
[品牌故事]
```

**使用提示：**
- 上传 5-8 张产品图片
- 提供 3-5 个核心卖点
- 填写完整的产品规格

### 模板 2：功能导向型

适合功能复杂的产品（电子产品、工具等）：

```
[品牌横幅]
    ↓
[主图 + 一句话卖点]
    ↓
[详细功能介绍 x5]
    ↓
[技术规格表]
    ↓
[使用教程]
    ↓
[常见问题]
```

### 模板 3：生活方式型

适合时尚、家居、美妆等产品：

```
[品牌横幅]
    ↓
[场景主图]
    ↓
[情感卖点]
    ↓
[多场景展示]
    ↓
[用户评价精选]
    ↓
[品牌理念]
```

## 优化技巧

### 关键词优化

OuraPix 会自动优化关键词布局：

1. **标题** - 包含核心关键词
2. **卖点** - 长尾关键词自然融入
3. **后端关键词** - 补充同义词、变体词

### 图片优化

- **主图**：纯白背景，产品占 85% 以上
- **场景图**：展示产品使用情境
- **信息图**：用图形展示产品特点
- **对比图**：与竞品或旧版对比

### 文案最佳实践

**标题公式：**
```
[品牌] + [核心关键词] + [关键特性] + [规格/数量]
```

**卖点撰写：**
- 每点聚焦一个特性
- 先讲好处，再讲特性
- 使用具体数据

## 常见问题

**Q: A+ 内容会影响搜索排名吗？**

A+ 内容本身不直接影响搜索排名，但通过提升转化率间接影响排名。

**Q: 可以修改已发布的 A+ 内容吗？**

可以，修改后需要重新提交审核，通常 1-2 个工作日完成。

**Q: 所有卖家都可以使用 A+ 内容吗？**

需要完成品牌注册（Brand Registry）才能使用 A+ 内容功能。
