# Shopify 详情页设计指南

Shopify 为独立站卖家提供了更大的自由度。本文介绍如何使用 OuraPix 创建品牌化的 Shopify 产品页面。

## Shopify 产品页结构

### 标准布局

```
[导航栏]
    ↓
[产品标题 + 价格]
[主图画廊]    [购买按钮]
              [变体选择]
              [数量选择]
              [加入购物车]
    ↓
[产品描述]
    ↓
[产品详情标签页]
    ↓
[推荐产品]
    ↓
[用户评价]
```

## 设计要素

### 色彩系统

OuraPix 支持自定义品牌色彩：

```css
/* 主色调 - 用于 CTA 按钮 */
--primary: #your-brand-color;

/* 辅助色 - 用于强调 */
--accent: #complementary-color;

/* 背景色 */
--background: #ffffff;
--surface: #f5f5f5;

/* 文字色 */
--text-primary: #1a1a1a;
--text-secondary: #666666;
```

### 字体选择

| 风格 | 标题字体 | 正文字体 |
|------|----------|----------|
| 现代 | Inter | Inter |
| 优雅 | Playfair Display | Source Sans Pro |
| 活力 | Poppins | Open Sans |
| 专业 | Montserrat | Lato |

### 图片规格

**主图：**
- 尺寸：2048x2048px
- 比例：1:1 或 4:5
- 格式：JPG/PNG

**详情图：**
- 宽度：1920px
- 高度：自适应
- 文件大小：< 500KB
